# rm-taller-git-2025

Este es un proyecto de demostración creado con el único propósito de practicar los comandos básicos de `git` y el flujo de trabajo de GitHub.

## Propósito

El objetivo principal es simular un entorno de desarrollo para aprender y experimentar con:

* Clonación de repositorios (`git clone`)
* Creación y cambio de ramas (`git branch`, `git checkout`)
* Confirmación de cambios (`git add`, `git commit`)
* Subida de cambios a GitHub (`git push`)
* Actualización desde el repositorio remoto (`git pull`)
* Revisión de historial (`git log`)

## Contenido

Este repositorio contiene archivos básicos (como este `README.md` y un `RUN.md`) que sirven como contenido de ejemplo para ser modificados.
